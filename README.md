# Dota 2 Match Tracker (Local)

Локальный сайт для ручного ведения статистики матчей Dota 2. Все данные хранятся в браузере (IndexedDB), работают оффлайн и не требуют бэкенда.

## Запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите локальный сервер разработки:

```bash
npm run dev
```

3. Откройте сайт в браузере:

```
http://localhost:5173
```

## Возможности

- Dashboard с краткой сводкой и последними матчами.
- Добавление матчей через форму с валидацией.
- Список матчей с фильтрами, поиском, сортировкой, редактированием и удалением.
- Аналитика: винрейт по героям, сложные оппоненты, лучшие союзники, тренды.
- Экспорт/импорт JSON, сброс данных, настройки.

## Локальные данные

Все матчи сохраняются в IndexedDB и не теряются при перезагрузке страницы. Для резервного копирования используйте экспорт JSON в разделе Settings.
